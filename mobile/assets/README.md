# App assets

Add the following for production builds:

- **icon.png** — 1024×1024 app icon
- **splash.png** — Splash screen image (e.g. 1284×2778). *Note: Design must include "Monet" title, `icon.png` in the center, and "powered by Jerttech" on a `#0D0D0D` background to match the JS loading screen.*
- **adaptive-icon.png** — Android adaptive icon foreground (1024×1024)

For local development, Expo may use defaults if these are missing.
